Prepared by: Abdu

Introduction

Ports are communication endpoints used by computers to send and receive data over a network. Each port is associated with a specific service or protocol. Understanding commonly used ports is essential in cybersecurity for both defense and attack analysis.

Simple Explanation

Ports = doors 🚪 of a computer

Each door is used for a specific service

Hackers scan ports to find open doors

Security professionals close or protect these doors

Important Notes

✔ Port 22 (SSH) → very important for secure access
✔ Port 80 vs 443 → HTTP vs HTTPS (secure)
✔ Port 21 (FTP) → insecure, often targeted
✔ Port 445 (SMB) → used in many attacks

🛡 Security Perspective
Open ports can be entry points for attackers
Use firewalls to block unnecessary ports
Always prefer secure protocols (HTTPS, SSH)

Conclusion

common ports helps in both network configuration and cybersecurity defense. Monitoring and securing open ports is essential to protect systems from attacks.
